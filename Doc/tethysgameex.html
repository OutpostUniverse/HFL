TethysGameEx Reference

Overview

TethysGameEx is an extension to the standard TethysGame class.

There are two ways to use TethysGameEx. You can continue to use it as normal, for example TethysGameEx::function:

TethysGameEx::ReloadSheets();

You can also use the global variable gGame to access it:

gGame.ReloadSheets();
