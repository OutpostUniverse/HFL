ActivateReport

ActivateReport loads a new report into the Command Pane.

void ActivateReport(PaneReport *report);

If report is NULL, the pane will be cleared.

Upon load, the report's Init virtual method will be called. At that time, the report should initialize all data and controls.

The old report's button will be popped up if the old report's GetLinkedButtonId virtual method returns a nonzero value.
