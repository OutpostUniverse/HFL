CreatorID

CreatorID returns the ID of the player who created the unit. 
int CreatorID();
Usually, but not always, the Creator ID of a unit is the same as its owner. However, the owner ID changes when a unit is captured or transferred to another player, unlike the creator ID.
The creator ID can be used to find out what upgrades and settings the unit uses (as the unit inherits upgrade info from the player who created it, not the current owner).
