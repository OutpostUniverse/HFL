SetSatelliteCount

SetSatelliteCount will set the number of 'satellites' of a specific type that the player has.

void SetSatelliteCount(map_id objectType, int count);
See the page on GetSatelliteCount for a discussion of what a satellite is and what values can be used for objectType.
If you give an invalid map_id for objectType, nothing will happen.

The following code gives a player 5 RLV's. These will be up in space and will be able to land at an empty Spaceport. 

ExtPlayer[0].SetSatelliteCount(mapRLV, 5);
