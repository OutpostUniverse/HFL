GameMapEx Reference

Overview

GameMapEx is an extension to the standard GameMap class.

There are two ways to use GameMapEx. You can continue to use it as normal, for example GameMapEx::function:

GameMapEx::GetMapHeight();

You can also use the global variable gMap to access it:

gMap.GetMapHeight();
