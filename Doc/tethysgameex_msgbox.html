MsgBox

MsgBox displays an OP2-style message box. 

int MsgBox(HWND hwnd, LPCTSTR lpText, LPCTSTR lpCaption, UINT uType);

See the MessageBox API in any Windows API reference to learn how to use the MsgBox method.

Keep in mind that the message box will be modal; that is, it will suspend the game. If used in multiplayer the player displaying the message box will lag out of the game.
