UnitInfo Reference

Overview

UnitInfo is a new class whose means is to access the internal UnitInfo classes inside the game. This allows you to set unit parameters without having to use UNIT_PROP or other means in a techtree. (It also allows setting of more variables than are available through techtrees).

To use UnitInfo, just declare a new UnitInfo var and pass either a map_id or a codename of a unit to it:

UnitInfo lynx(mapLynx);
// or ...
UnitInfo lynx("LIGHT_TANK");

Note that many of the methods for UnitInfo are just aliases for each other. This is because the game stores data in the same locations for different types of units, but interprets the values differently.
