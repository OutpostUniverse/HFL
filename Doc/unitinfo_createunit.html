CreateUnit

CreateUnit creates a new unit instance from the current UnitInfo class and returns the instance. 

Unit CreateUnit(LOCATION where, int unitId);

where specifies a unit location in pixels. If unitId is nonzero, the unit with that ID will be overwritten (make sure that the old unit is dead first). The unit count will still be incremented even if an ID is specified.

This function is for power users who want extra control over the unit. Everyone else should use TethysGame::CreateUnit to create a unit.
