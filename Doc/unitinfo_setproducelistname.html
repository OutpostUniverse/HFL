SetProduceListName

SetProduceListName sets the name that will appear in factory build menus. (For example Factory, Structure is used instead of Structure Factory). 
void SetProduceListName(char *newName);

OP2 imposes a limit of 40 bytes for the name string. If your string is longer than 39 chars (39 + null = 40 bytes) the produce list name will not be set.
