SetCheatUnlimitedResourcesEx

SetCheatUnlimitedResourcesEx enables infinite resources for all players. It is a replacement for SetCheatUnlimitedResources. 

void SetCheatUnlimitedResourcesEx(int boolOn);

This function enables test code built into the game, it should be limited to debugging use only as it gives players a 'cheat' advantage (although it takes effect for all players).
