TriggerEx Reference

Overview

TriggerEx is an extension to the OP2 Trigger class. Its main features are that it allows you to change trigger data and parameters after the trigger has been created.
TriggerEx is derived from Trigger, so if you use TriggerEx in place of Trigger when you define variables, you'll be able to use the new functions.
