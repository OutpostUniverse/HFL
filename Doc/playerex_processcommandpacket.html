ProcessCommandPacket

ProcessCommandPacket injects a game command packet directly into the game engine locally to be processed. (The packet will not get sent over the network). 
int ProcessCommandPacket(CommandPacket *packet);
No error checking or input validation is performed on the packet. The result from the game engine is returned (typically a true or false value).

This function is intended only for those familiar and comfortable with using CommandPackets directly.
Unwanted behavior will almost certainly result if invalid data is passed.
