UnitEx Reference

Overview

UnitEx is an extension to the OP2 Unit class. It allows you to issue various commands that are not possible with the standard Unit functions, as well as retrieve extra information about the unit.
UnitEx is derived from Unit, so anywhere you declare Unit variables, you can declare UnitEx variables and use them in the same way. You can still call functions from the normal Unit class on them as well. For example, the following is perfectly legal code: 
UnitEx scout;
TethysGame::CreateUnit(scout, mapScout, LOCATION(32+31, 32-1), 0, mapNone, 0);

int owner = scout.OwnerID(); // call Unit function
int creator = scout.CreatorID(); // call UnitEx function
