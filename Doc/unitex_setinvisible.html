SetInvisible

SetInvisible sets whether a structure will be invisible. 
void SetInvisible(int boolOn);
If boolOn is nonzero, the invisible flag will be set, otherwise it will be cleared.
Structures with the invisible flag will not be drawn and the player cannot select them (however they are still visible on the Minimap).

The invisible flag is for structures only. The game may crash or behave incorrectly if you set the invisible flag on a vehicle.
