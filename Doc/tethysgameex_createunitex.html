CreateUnitEx

CreateUnitEx calls an internal unit creation function, allowing you to create objects that typically would crash the game when created normally. 
int CreateUnitEx(map_id unitType, int pixelX, int pixelY, int creatorId, int cargoWeapon, int unitIndex, int boolCenterInTile);

unitType specifies the map_id of the unit to create.
pixelX and pixelY specify the new unit's location in pixels (multiply tile positions * 32 to get pixels)
creatorId specifies the player who creates and initially owns the unit.
cargoWeapon specifies the cargo or weapon type to use.
unitIndex specifies the existing unit ID to overwrite if nonzero (make sure the old unit is dead first)
boolCenterInTile centers the unit on the tile it's placed over if nonzero.

CreateUnitEx can be useful to create EMP blasts on the map, as well as "Pretty art!" / AMBIENT_ANIM type objects.

This function is intended for power users only, as various different checks on the input parameters are skipped. If you aren't using this function for a specific reason you should use TethysGame::CreateUnit instead to create units.
