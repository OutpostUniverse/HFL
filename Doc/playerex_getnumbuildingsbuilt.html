GetNumBuildingsBuilt

GetNumBuildingsBuilt returns the number of buildings constructed by the player.

int GetNumBuildingsBuilt();
GetNumBuildingsBuilt can be used to control AI and other tasks, for example after the player has 10 buildings the AI will start attacking: 
if (ExtPlayer[0].GetNumBuildingsBuilt() >= 10)
{
	// start the attacks from the AI base here... :-)
}
